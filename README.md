# my.nvim

## Installation

> With [packer.nvim](https://github.com/wbthomason/packer.nvim)

Add to your `plugins.lua`, directly in your `init.nvim` (within a Lua group), or `init.lua`:

```lua
use({'jbaxx/my.nvim'})
```

To install:
```
:PackerInstall
```

## Usage

To display the mappings in the help window:
```
:h my.nvim
```
